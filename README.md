### Funtasya World - Laravel CRUD Project Internship Test

# How To Run

1. composer install
2. npm install\
3. cp .env.example .env
4. php artisan key:generate
5. create database tokokita in mysql
6. php artisan migrate
7. npm run dev (Terminal 1)
8. php artisan serve (Terminal 2)
9. akses halaman di http://127.0.0.1:8000/
